class EDA.EDAProcessor(target, ordinal_columns, exclude_columns=None, verbose=True)[fuente]

Bases: object

Clase para procesar tareas de análisis exploratorio de datos (EDA).

Parámetros:

  • target (str) – Nombre de la columna objetivo.

  • ordinal_columns (list) – Lista de columnas ordinales.

  • exclude_columns (list | None (default=None)) – Columnas a excluir del análisis. Por defecto, None.

  • verbose (bool (default=True)) – Indica si se imprimen mensajes durante el procesamiento. Por defecto, True.

Variables:

  • dict_column (dict) – Diccionario para almacenar metadatos de las columnas.

  • n_samples (int) – Número de filas en el conjunto de datos.

  • n_columns (int) – Número de columnas en el conjunto de datos.

  • column_names (list) – Lista con los nombres de las columnas.

  • categorical_features (list) – Lista de columnas categóricas.

  • numerical_features (list) – Lista de columnas numéricas.

distribution_variable(data, column, n_bins=None, bins=None)[fuente]

Calcula la distribución de una variable dividiéndola en intervalos.

Este método permite dividir los valores de una columna en intervalos (bins) y calcular la frecuencia de valores dentro de cada intervalo.

Parámetros:

  • data (pd.DataFrame) – Conjunto de datos que contiene la columna a analizar.

  • column (str) – Nombre de la columna para la cual calcular la distribución.

  • n_bins (int, optional (default=None)) – Número de intervalos (bins) en los que dividir la columna. Si se proporciona, los intervalos se calcularán automáticamente usando valores equiespaciados.

  • bins (list[float] | list[int], optional (default=None)) – Lista de límites de los intervalos definidos manualmente. Si se proporciona, n_bins será ignorado.

Devuelve:

Serie con la frecuencia de valores en cada intervalo, ordenada por los intervalos.

Tipo del valor devuelto:

pd.Series

Muestra:

ValueError – Si el nombre de la columna no está en el conjunto de datos.

get_results()[fuente]

Obtiene el resumen del análisis exploratorio de datos (EDA).

Este método devuelve un DataFrame que contiene metadatos de cada columna en el conjunto de datos, incluyendo información como:

  • Tipo de dato.

  • Número de valores faltantes.

  • Porcentaje de valores faltantes.

  • Número de valores únicos.

  • Si la columna tiene filas duplicadas.

Devuelve:

Un DataFrame donde cada fila corresponde a una columna en el conjunto de datos, y los metadatos de cada columna se almacenan en la fila respectiva.

Tipo del valor devuelto:

pandas.DataFrame

plot_categorical_variables(data, cat_vars=None)[fuente]

Genera gráficos de barras para las variables categóricas seleccionadas.

Este método crea gráficos de barras para analizar la distribución de las variables categóricas en el conjunto de datos. Si se seleccionan múltiples variables, se generarán subgráficos.

Parámetros:

  • data (pd.DataFrame) – Conjunto de datos que contiene las variables categóricas a graficar.

  • cat_vars (list[str], optional (default=None)) – Lista de nombres de columnas categóricas a graficar. Si no se proporciona, se utilizarán todas las variables categóricas identificadas.

Devuelve:

Este método no retorna ningún valor. Genera gráficos directamente.

Tipo del valor devuelto:

None

Muestra:

ValueError – Si alguna de las variables proporcionadas en cat_vars no se encuentra en las variables categóricas identificadas.

run(data)[fuente]

Ejecuta el análisis exploratorio de datos (EDA).

El EDA realiza los siguientes pasos:

  1. Identifica el tipo de variable (numérica, categórica, etc.).

  2. Calcula valores faltantes y duplicados.

  3. Calcula la matriz de correlación.

Parámetros:

data (pandas.DataFrame) – Conjunto de datos a analizar.

Muestra:

TypeError – Si el argumento data no es un DataFrame de pandas.
