Outliers

class Outliers.OutliersDetection(method, k=1.5, threshold=0.05, seed=42, estimator=None, contamination=0.05, n_neighbors=20, n_components=2)[fuente]

Bases: object

Clase para la detección de valores atípicos (outliers) en variables numéricas y categóricas.

Esta clase implementa múltiples métodos para detectar valores atípicos en conjuntos de datos, incluyendo técnicas estadísticas y modelos basados en machine learning.

Parámetros:

  • method (str) –

    Método para detectar valores atípicos. Puede ser uno de los siguientes:

    • IQR: Rango intercuartílico.

    • std: Desviación estándar.

    • IForest: Isolation Forest.

    • LOF: Local Outlier Factor.

    • GMM: Gaussian Mixture Model.

    • all: Aplica todos los métodos disponibles.

  • k (float, optional (default=1.5)) –

    Valor multiplicador utilizado en los métodos estadísticos (IQR o std).

    • Si method es IQR, los límites se calculan como:

    \[\begin{split}\text{lower limit} = Q_1 - k \cdot (Q_3 - Q_1) \\ \text{upper limit} = Q_3 + k \cdot (Q_3 - Q_1)\end{split}\]

    • Si method es std, los límites se calculan como:

    \[\begin{split}\text{lower limit} = \mu - k \cdot \sigma \\ \text{upper limit} = \mu + k \cdot \sigma\end{split}\]

  • threshold (float, optional (default=0.05)) – Umbral para identificar valores atípicos en variables categóricas, basado en la frecuencia relativa.

  • seed (int, optional (default=42)) – Semilla para los métodos que requieren aleatoriedad.

  • estimator (object, optional (default=None)) – Estimador personalizado. Puede ser una instancia de IsolationForest, LocalOutlierFactor o GaussianMixture.

  • contamination (float, optional (default=0.05)) – Proporción de datos considerados como valores atípicos para los métodos basados en estimadores.

  • n_neighbors (int, optional (default=20)) – Número de vecinos para el método Local Outlier Factor.

  • n_components (int, optional (default=2)) – Número de componentes en el modelo Gaussian Mixture.

Variables:

  • numerical_features (list) – Lista de nombres de variables numéricas en el conjunto de datos.

  • categorical_features (list) – Lista de nombres de variables categóricas en el conjunto de datos.

  • dict_column (dict) – Diccionario con los resultados de la detección de outliers en variables numéricas.

  • dict_column_cat (dict) – Diccionario con los resultados de la detección de outliers en variables categóricas.

combine_categories(data, var, rare_categories, new_category)[fuente]

Combina categorías poco frecuentes en una categoría existente o nueva.

Parámetros:

  • data (pandas.DataFrame) – Conjunto de datos original.

  • var (str) – Nombre de la variable categórica.

  • rare_categories (list) – Lista de categorías que se combinarán.

  • new_category (str) – Nombre de la nueva categoría que representará las categorías combinadas.

Devuelve:

DataFrame modificado con las categorías combinadas.

Tipo del valor devuelto:

pandas.DataFrame

get_outliers(variable)[fuente]

Obtiene la lista de índices de valores atípicos para una variable específica.

Parámetros:

variable (str Parameters) – Nombre de la variable de la cual se quieren obtener los índices de valores atípicos.

Devuelve:

Lista de índices de los valores atípicos.

Muestra:

ValueError – Si la variable no ha sido analizada previamente.

get_results(method=None)[fuente]

Devuelve un resumen de los valores atípicos detectados para variables numéricas.

Parámetros:

method (str (default=None)) – Método específico para el cual se quieren los resultados. Si no se especifica, se devolverán resultados de todos los métodos.

Devuelve:

Resumen de los valores atípicos detectados por variable y método.

Tipo del valor devuelto:

pd.DataFrame

get_results_cat_vars()[fuente]

Devuelve un resumen de los valores atípicos detectados en variables categóricas.

Devuelve:

Resumen de los valores atípicos detectados para cada variable categórica.

Tipo del valor devuelto:

pd.DataFrame

run(data)[fuente]

Ejecuta el análisis de detección de valores atípicos.

Parámetros:

data (pandas.DataFrame) – Conjunto de datos sobre el que se detectarán valores atípicos.

transform(data, method, metric='median')[fuente]

Sustituye los valores atípicos detectados con la mediana o la media.

Parámetros:

  • data (pandas.DataFrame) – Conjunto de datos original.

  • method (str) – Método utilizado para identificar los valores atípicos.

  • metric (str | dict, optional) – Puede ser «median» o «mean» para aplicar el mismo valor a todas las variables, o un diccionario para aplicar una métrica diferente a cada variable. Por defecto, «median».

Devuelve:

Nuevo DataFrame con los valores atípicos sustituidos.

Tipo del valor devuelto:

pandas.DataFrame
