class Preprocessing.Preprocessing(target, missing_strategy_num='median', missing_strategy_cat='most_frequent', fill_value_num=None, fill_value_cat=None, categorical_encoding='onehot', scaler_method='minmax')[fuente]

Bases: object

Clase para preprocesar datos numéricos y categóricos.

Esta clase implementa estrategias para imputar valores faltantes, codificar variables categóricas y escalar variables numéricas.

Parámetros:

  • target (str) – Nombre de la variable objetivo que no será transformada.

  • missing_strategy_num (str (default="median")) – Estrategia de imputación para características numéricas. Puede ser «mean», «median», «most_frequent» o «constant».

  • missing_strategy_cat (str (default="most_frequent")) – Estrategia de imputación para características categóricas. Puede ser «most_frequent» o «constant».

  • fill_value_num (float | int, optional (default=None)) – Valor constante utilizado para imputar valores numéricos si missing_strategy_num es «constant».

  • fill_value_cat (str, optional (default=None)) – Valor constante utilizado para imputar valores categóricos si missing_strategy_cat es «constant».

  • categorical_encoding (str (default="onehot")) – Método de codificación para características categóricas. Puede ser «onehot», «label» o «dummies».

  • scaler_method (str (default="minmax")) – Método para escalar variables numéricas. Puede ser «minmax», «standard» o «robust».

Variables:

  • num_imputer (sklearn.impute.SimpleImputer) – Instancia de SimpleImputer utilizada para imputar valores numéricos.

  • cat_imputer (sklearn.impute.SimpleImputer) – Instancia de SimpleImputer utilizada para imputar valores categóricos.

  • encoder (sklearn.preprocessing.OneHotEncoder | dict) – Codificador utilizado para variables categóricas. Puede ser una instancia de OneHotEncoder o un diccionario de LabelEncoder.

  • scaler (sklearn.preprocessing.MinMaxScaler | sklearn.preprocessing.StandardScaler | sklearn.preprocessing.RobustScaler) – Escalador utilizado para normalizar variables numéricas.

fit_transform(data, numerical_columns=None, categorical_columns=None)[fuente]

Ajusta y transforma los datos.

Este método realiza imputación de valores faltantes, codificación de variables categóricas y escalado de variables numéricas en el conjunto de datos proporcionado.

Parámetros:

  • data (pandas.DataFrame) – Conjunto de datos de entrada.

  • numerical_columns (list[str], optional (default=None)) – Lista de nombres de columnas numéricas. Si es None, las columnas numéricas serán detectadas automáticamente.

  • categorical_columns (list[str], optional (default=None)) – Lista de nombres de columnas categóricas. Si es None, las columnas categóricas serán detectadas automáticamente.

Devuelve:

DataFrame transformado, con valores imputados, variables categóricas codificadas y variables numéricas escaladas.

Tipo del valor devuelto:

pandas.DataFrame
